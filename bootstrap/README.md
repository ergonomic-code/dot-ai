# Bootstrap templates

This directory contains templates used when integrating the framework into a target repository.
Copy these templates into the target repository and adapt them locally.

## Index

- [`AGENTS.md`](AGENTS.md) — Starter agent rules for the target repository.
- [`APPLICATION-CONTEXT.md`](APPLICATION-CONTEXT.md) — Starter application context template.
- [`SYSTEM-CONTEXT.md`](SYSTEM-CONTEXT.md) — Starter system and workspace context template.
